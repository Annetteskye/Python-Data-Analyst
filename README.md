# Анализ эффективности маркетинга и юнит-экономики (Python / Pandas)
📌 О проекте
Цель проекта — оценка рентабельности товарных категорий и эффективности рекламных вложений (ROAS) на базе транзакционных данных. Этот инструмент позволяет быстро выявлять убыточные направления и оптимизировать маркетинговый бюджет.
🛠 Стек технологий
Python (Pandas): Обработка данных, расчет производных метрик (Margin, ROAS, Profit).
Matplotlib: Визуализация результатов и построение бизнес-графиков.
Jupyter Notebook: Формат интерактивного отчета с кодом и выводами.
📈 Ключевые показатели (KPI)
В ходе анализа были рассчитаны следующие метрики:
Gross Profit (Валовая прибыль): Выручка за вычетом себестоимости (COGS).
Margin % (Маржинальность): Оценка доходности каждой категории.
ROAS (Return on Ad Spend): Окупаемость рекламных расходов. Позволяет понять, сколько выручки приносит каждый вложенный в рекламу рубль.
🖼 Визуализация
Окупаемость рекламы (ROAS) по категориям
Основные выводы:
Лидеры роста: Категории «Сумки» и «Платья» демонстрируют самый высокий ROAS (>6.0), значительно превышая целевой показатель (5.0).
Зона риска: Категория «Аксессуары» находится на грани окупаемости (ROAS 5.3). Рекомендовано пересмотреть рекламную стратегию или снизить стоимость привлечения в данном сегменте.
Оптимизация бюджета: Перераспределение бюджета с низкоэффективных (аксессуаров) на высокодоходные (сумки) позволит увеличить общую чистую прибыль без роста общих затрат.
🚀 Реализованный функционал:
Импорт и предобработка данных (Data Cleaning).
Автоматизированный расчет финансового результата по категориям.
Построение сравнительной столбчатой диаграммы с контрольной линией (Target Line).
Выполнение базовой дескриптивной статистики (describe()) для оценки распределения данных.
